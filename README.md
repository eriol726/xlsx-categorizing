C# application that categorize and summarize xlsx data downloaded from SEB's internet bank and store it in google drive.

#
Insert the google spreadsheet id and your google account username into value params in `App.config` :
````
<appSettings>
  <add key="spreadsheetId" value=""/>
  <add key="userName" value=""/>
</appSettings>
````
